1. Crie um arquivo HTML chamado "perfil.html".

2. Dentro do `<body>` do HTML, crie um cabeçalho `<header>` com um título "Meu Perfil".

3. Abaixo do cabeçalho, crie uma `<section>` para mostrar informações pessoais, como nome, foto e uma breve descrição.

4. Utilize tags semânticas como `<h1>`, `<p>`, `<img>` e `<div>` para estruturar o conteúdo.

5. No CSS (crie um arquivo "perfil.css"), estilize o fundo da página, as cores do texto e defina margens e espaçamentos para os elementos.

Dentro da pasta do projeto crie uma pasta css para os estilos e uma pasta img para as imagens.
